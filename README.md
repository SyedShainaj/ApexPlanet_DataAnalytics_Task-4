# Data Storytelling & Statistical Validation

## Overview

This project was completed as part of the Data Analytics Internship at ApexPlanet Software Pvt. Ltd. The objective was to analyze sales data, create meaningful visualizations, validate insights using statistical methods, and provide business recommendations.

## Objectives

* Analyze sales performance across different regions.
* Identify key business trends and patterns.
* Create visualizations to support decision-making.
* Perform hypothesis testing for statistical validation.
* Present insights through storytelling and dashboards.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Plotly
* SciPy
* Streamlit

## Key Analyses

* Regional Sales Analysis
* Category-wise Sales Performance
* Customer Segment Analysis
* Sales Trend Visualization
* Hypothesis Testing (ANOVA)

## Key Insights

* Identified top-performing regions and categories.
* Compared sales performance across different groups.
* Validated business insights using statistical testing.
* Generated actionable recommendations for business growth.

## Outcome
This project enhanced my skills in data analysis, visualization, statistical validation, and data-driven decision making.
